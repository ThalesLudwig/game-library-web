# Games Library

A React application to list Games using Typescript.
<br>
Data is provided by a test API from [rawg.io](https://rawg.io/).

Tools used:
- create-react-app
- typescript
- redux-toolkit
- react-router

## Project setup
- Local
```
$ npm install
$ npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
